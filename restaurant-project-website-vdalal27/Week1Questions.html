Week #1 Questions and Answers

HTML Question

What is the difference between !DOCTYPE html tag and html tag? how many html tags can be there in one document?

!DOCTYPE html tag is used to specify the version of html we want to use. By providing this tag, we tell the browser that we are using HTML5. While html tag works as a container. All the html code must be inside html tag, so there can only be one html tag.

CSS Question

Types in which CSS can be integrated and which one do you prefer?

There are 3 ways in which CSS can be integrated:

  • Inline: Style attribute applied to HTML elements individually
  • Embedded: Style attribute applied in the head element of the document
  • Imported: Link is provided to the external file containing the style

When we want to apply style to particular element Inline style can be useful but usually it is tiring process and the website turns out to be non symmetrical and not good looking.
When there is only one page to style, Embedded can be useful. But if we want to apply same style to multiple pages, this technique fails.
When we have a big website to style in similar way, Imported is very useful. We just need to provide the link to the file in our html document instead of adding same style to every html document.
